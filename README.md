# 🚀 ForgeX - Learning Platform

ForgeX is a modern learning platform developed as part of my SkillCraft Technology Internship (Task 3).

It helps students improve their programming skills through quizzes, progress tracking, and detailed performance reviews.

---

## ✨ Features

- 🔐 Multi-user Login & Registration
- 👤 Student Dashboard
- 🎯 Randomized Quiz System
- ⏳ 30 Minute Quiz Timer
- 📊 Result Analysis
- 📈 Best Score Tracking
- 📝 Quiz History
- 🔍 Answer Review System
- 💾 Local Storage Data Persistence
- 🌟 Responsive UI
- 🎨 Modern Dark Theme with Neon Effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
